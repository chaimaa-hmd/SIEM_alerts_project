## Description
Ce projet aide les analystes SOC à gérer un grand nombre d'alertes en les classant selon leur criticité.  
Les alertes sont générées de manière synthétique, et un modèle Random Forest prédit le score de criticité.  

## Fonctionnalités
- Génération synthétique de 500 alertes avec les attributs : source, utilisateur, fréquence et criticité.  
- Modèle Random Forest pour prédire la criticité des alertes.  
- Interface console permettant de définir un seuil dynamique pour afficher uniquement les alertes importantes.  
- Tri des alertes par score prédictif décroissant.
